## 🚀 Overview

BugLens is a website analysis platform that helps developers, businesses, and website owners discover and fix issues quickly.

The platform scans websites and provides insights on:

* 🐛 Bugs & Errors
* ⚡ Performance Issues
* 🔒 Security Weaknesses
* 📈 SEO Problems
* ♿ Accessibility Issues
* 🔗 Broken Links
* 📱 Mobile Compatibility
* 🌐 Best Practice Violations

## 📦 Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/Bug-Analyzer.git
cd Bug-Analyzer
```

### Install Frontend Dependencies

```bash
npm install
```

### Install Backend Dependencies

```bash
cd backend
npm install
```

---

## ⚙️ Environment Variables

Create a `.env` file:

```env
VITE_API_URL=YOUR_BACKEND_URL
```

Example:

```env
VITE_API_URL=https://your-backend-url.com
```

---

## ▶️ Run Locally

### Frontend

```bash
npm run dev
```

### Backend

```bash
cd backend
npm start
```
